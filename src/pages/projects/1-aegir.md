A real-time, full-stack management platform for student registration, event sequencing, and automated official document generation for swimming meets.

- **Real-time Sync**: Rapid data entry with instant **Convex** synchronization across all devices.
- **Smart Management**: Features `dnd-kit` for event reordering and rules validation for student entries.
- **Automated Reporting**: High-fidelity PDF generation of meet programs and registration sheets via **Puppeteer**.
- **Modern Architecture**: Built with **Next.js 15**, **Radix UI**, and **Tailwind CSS** for a premium, mobile-optimized experience.
