The official web portal for the **Sri Lanka University Sports Association (SLUSA)**, designed to manage and showcase national university sports data and historical achievements.

- **Dynamic Results Management**: Developed a system using **Astro** and **React** to display real-time and archived sports results, including data from the 3rd and 5th Sri Lanka University Games (SLUG).
- **Historical Data Archiving**: Integrated **MDX** for structured and efficient processing of legacy news articles and event archives.
- **Enhanced UI/UX**: Implemented responsive layouts with **Tailwind CSS**, featuring interactive `NewsArticleCard` components and optimized navigation to improve stakeholder engagement.
- **Performance Optimization**: Focused on reducing cumulative layout shifts (CLS) and optimizing asset delivery for a faster user experience.
