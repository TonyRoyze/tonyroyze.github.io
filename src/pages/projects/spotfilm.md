A premium, curated online gallery designed for vintage treasures and timeless artifacts, emphasizing a clean and aesthetic user experience.

- **Astro-Powered Performance**: Built using **Astro** for lightning-fast, content-driven performance, achieving a Lighthouse score of 90+.
- **Modern Responsive Design**: Leveraged **Tailwind CSS** to implement mobile-first layouts and refined UI components, including specialized `LocationCard` for physical storefront details.
- **SEO & Social Optimization**: Integrated comprehensive Open Graph meta tags and schema markup to ensure professional search engine presence and social media engagement.
- **High-Impact Visuals**: Designed an immersive `Hero` section and curated item grids tailored to showcase high-value antiques with minimal distraction.
