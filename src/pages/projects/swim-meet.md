A robust, full-stack competition management platform engineered to handle real-time results and coordination for interfaculty swimming meets.

- **Real-Time Leaderboards**: Built with **Next.js** and **Redux** for centralized state management, enabling fluid, real-time updates of event standings and faculty rankings.
- **Advanced State Management**: Leveraged **Redux** to ensure data synchronization across complex UI components, handling high-frequency data inputs during live events.
- **Premium Interface**: Utilized **Shadcn UI** and **Radix UI** primitives to create a high-quality, accessible, and themeable interface with professional toast notifications and loading states.
- **Secure Management Dashboard**: Developed a specialized dashboard for officials to manage heat assignments, results verification, and faculty filtering with optimized backend communication.
