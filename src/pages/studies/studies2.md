A comprehensive foundation in modern software development and systems engineering, covering the full software development lifecycle (SDLC). 

- **System Design & Implementation**: Developed proficiency in designing scalable system architectures and implementing them using industry-standard practices.
- **Full-Stack Web Development**: Gained extensive experience in both frontend and backend technologies, focusing on creating dynamic and responsive web applications.
- **Database Management Systems**: In-depth study of relational database design, SQL, and data integrity management (RDBMS).
- **Networking & Communications**: Understanding the fundamentals of network protocols, security, and infrastructure management.
- **Mobile Application Development**: Designed and built cross-platform mobile solutions, prioritizing user-centric design (UI/UX).
