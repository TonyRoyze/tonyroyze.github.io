Aayu Technologies is a specialized software firm providing enterprise-grade B2B messaging and file transfer solutions. During my internship, I contributed to their cloud-native ecosystem, which includes:

- **MFT Gateway**: A world-class SaaS platform for secure B2B file transfers using the AS2 protocol, built on a serverless architecture.
- **AS2 Gateway**: A secure document exchange engine ensuring data integrity and non-repudiation in business transactions.
- **EDI Generator**: A tool designed to simplify the creation and management of standard EDI documents like Purchase Orders and Invoices.

### Key Responsibilities & Learning:
- **Frontend Development (Angular)**: Enhanced the management dashboards for the gateways, focusing on real-time transfer monitoring and trading partner configuration. Implemented UI refinements to improve responsiveness and reduce layout shifts (CLS).
- **Backend Development (Java Spring Boot)**: Gained experience in building scalable backend services for B2B data processing. Worked on core logic for document parsing and secure communication protocols.
- **Cloud-Native Solutions**: Explored the integration of serverless technologies (AWS) in building cost-effective and scalable B2B communication software.
- **Optimization & Documentation**: Identified UI bottlenecks and documented optimization strategies, contributing to the overall performance of the web applications.
